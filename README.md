# Day-3-responsive-navbar-2# Responsive Navigation Bar (HTML, CSS & JavaScript)

A modern and fully responsive navigation bar built using **HTML, CSS, and JavaScript**.  
This version includes a mobile hamburger menu and dynamic background behavior for better user experience.

---

## Features

- Fully responsive design (desktop, tablet, mobile)
- Hamburger menu for small screens
- Dynamic background on scroll
- Smooth transitions and animations
- Clean and modern UI

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox, Media Queries, Transitions)  
- JavaScript (DOM manipulation)

---

---

## How It Works

### HTML
- Defines the structure of the navbar and menu items.

### CSS
- Handles layout using Flexbox.
- Adds responsiveness using media queries.
- Adds transitions and background styling.

### JavaScript
- Toggles the hamburger menu.
- Adds background change on scroll.
- Handles user interactions.
